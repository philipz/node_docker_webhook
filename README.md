# node_docker_webhook
#########################################
Just like [Go version Dockerhub webhook](https://github.com/cpuguy83/dockerhub-webhook-listener), this is NodeJS version.
And integrate [Docker Machine](https://github.com/docker/machine) and [Microsof Azure](http://azure.microsoft.com).

## Usage
```
npm install http node-simple-router dockerode sh simple-xmpp
node server.js
```
